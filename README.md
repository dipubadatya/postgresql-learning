# PostgreSQL Learning Journey

A repository documenting my daily PostgreSQL learning through hands-on SQL practice, examples, and exercises. Each folder represents a day's progress and contains the SQL scripts covered on that day.

## Overview

This repository is intended to:

* Learn PostgreSQL from fundamentals to advanced concepts.
* Practice SQL through real examples.
* Build a structured reference for future revision.
* Track daily learning progress consistently.
